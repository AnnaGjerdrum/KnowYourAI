Object recognition - Image Classificaion Algorithm


Here you will learn the details about how the algorithm for image classification works.
This algorithm can be found in products already on the market, such as the face finder on your camera or automatic trash sorting



Start from the start

To learn how this image classification works we will need to understand convolutional neural network. But even before that, it is important to know how the traditional classification approach works.

When you see this image, what does your brain tell you immediately?


Features

If I guessed it right what your brain would have told you is that it is a dog. How does our brain do that? How do you decide it is a dog and not a human or a cat or a car?

The answer is simple, the picture has a round nose, furry hair, long tongue, marble-like eyes which are unique combinations to be in one picture and so your brain immediately told to you it is a dog.



These attributes that I mentioned is called “features”. In a traditional approach, humans use to handpick these features and then plot it in a multidimensional space and try to find an optimal hyperplane to classify different classes. Some of the classification algorithms used were namely Support vector machines, Neural networks, etc. Selecting these features and describing them in the right way was a tedious process.


Layers in a neural network

After the invention of the neural networks in 1979 followed by AI winter, the Neural network was again looked into deeply in the 1990’s. Initially, during this period it was used to solve simple problems like number recognition and character recognition.


A neural network is built from 3 types of layers. The input layer is the initial data. In the traditional classification, the original image is processed and a feature vector is produced which represents the image. This feature vector will be, say 10-dimensional input to this neural network. Hidden layers are in between a fully connected layer. This is where the network learns the input to classify. You can say by optimizing these parameters the classifier is prepared. The output layer produces the result for the given inputs.

Advancement of Neural Networks

Around 2005 onwards the cost of GPU started to reduce and got much powerful. This helped in the advancement of the neural network and CNN (Convolutional neural network) was introduced. This revolutionalized the way we were doing classification. One big change this brought was to remove the feature extraction step totally as a pre-processing and include it as part of the neural network.




Automization of features

This means now the input to the neural network was the whole image and the network itself will extract features and learn the class. So it was basically optimizing the feature extraction. Now the neural network itself will figure out what it is seeing and why it is a Dog. This very similar to how humans learn. CNN is one of the networks. Similarly, there are other networks also like AlexNet (One of the oldest), LeNet, ResNet, ACGAN, Yolo, etc. To understand how these networks extract the features, below I have shown an image of the network.

Process of training a classification model

In supervised Machine learning, there are 2 phases in classification

  • Training phase: Here the ML algorithm takes the image dataset and trains the ML model like CNN, RNN to learn for the corresponding labels.
  • Prediction phase: In the prediction phase the trained model is taken and the images are passed to the classifier model to predict the output label for which the model is trained.
